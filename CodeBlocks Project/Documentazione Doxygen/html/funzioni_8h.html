◆ acquisizione()

ListaPtr * acquisizione ( )

Svolge la funzionalita' di dominio dell'acquisizione. Apre un file e salva i dati in una struttura dinamica gestita come lista.

Returns
NULL se l'acquisizione non e' possibile, altrimenti ritorna il puntatore alla lista concatenata creata.
Precondition
il file deve esistere, deve essere aperto correttamente, non deve essere vuoto e deve essere nel formato corretto.
Postcondition
lista concatenata non vuota.

Se file non aperto correttamente, restituisci NULL;

Se file vuoto, restituisci NULL;

Se file non formattato correttamente, restituisci NULL;

Creazione struttura dinamica gestita come lista, creazione del tipo di dato astratto Lista;
