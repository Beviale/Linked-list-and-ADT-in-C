Detailed Description

Implementazione della libreria riguardante la gestione dei file.

Authors
Alessandro Bevilacqua, Michele Como, Vito De Tommaso
Version
1.0
Date
2022-07-27 Con questa libreria possiamo aprire, chiudere, leggere e scrivere file ad accesso sequenziale. L'acquisizione dei dati avviene mediante l'uso di una struttura dinamica a lista concatenata.

apre un file in modalità lettura.

Suddivide ogni riga del file in token in occorrenza del carattere '&' e salva i campi in una struttura dinamica gestita come lista.

Returns
file aperto oppure puntatore NULL se l'operazione non è andata a buon fine.

Suddivide ogni riga del file in token in occorrenza del carattere '&' e salva i campi in una struttura dinamica gestita come lista.

Precondition
lista vuota e file aperto correttamente.
Postcondition
lista piena.

Stringa relativa ad ogni riga del file (quindi ad ogni corso).

Riga del file suddivisa in occorenza del carattere '&'.

Conta il numero di token suddivisi per ciascuna riga.

PROGRAMMAZIONE DIFENSIVA CLASSE ARRAY: nomecorso deve avere lunghezza massima SIZE.
